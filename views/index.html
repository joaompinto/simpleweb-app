The Simple Web App Development Framework

SimpleWeb is a Python web framework in early development. It provides a Model-View-Controller engine by integrating some of the best python components for each function. The HTTP/Web Application engine is built on top of CherryPy. The view/rendering uses the Mako templating library. The data modeling component is still being defined. The sample applications use the HTML, CSS, and JS Boostrap framework.

It was packages/tested for the following Platform as a Service (PaaS) cloud offerings:

  • Google Application Engine
